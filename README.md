# Currículo Fácil

Gerador de currículos desenvolvido em Python utilizando Flask e ReportLab.

## Funcionalidades

- Dados pessoais
- Experiência profissional
- Formação
- Certificações
- Habilidades
- Exportação em PDF

## Tecnologias

- Python
- Flask
- ReportLab

## Como executar

```bash
pip install -r requirements.txt
python app.py
```
